ShadowGuard Pro

La solution logicielle de protection contre les attaques ROP pour les applicatifs Linux userland

1. Introduction

ShadowGuard Pro est une solution innovante de sécurité informatique qui implémente une Shadow Stack logicielle pour protéger les applicatifs Linux userland contre les attaques de type Return-Oriented Programming (ROP). En surveillant et en validant les adresses de retour des fonctions, ShadowGuard Pro empêche les attaquants d'exploiter les vulnérabilités mémoire pour exécuter du code malveillant.

2. Fonctionnement Technique

2.1. Principe de la Shadow Stack

La Shadow Stack est une pile parallèle qui stocke les adresses de retour des fonctions de manière sécurisée. Contrairement à la pile standard, qui peut être corrompue par des attaques, la Shadow Stack est protégée et ne peut être modifiée que par des instructions spécifiques.

2.2. Mécanisme de Protection

  • Validation des Adresses de Retour : À chaque retour de fonction, ShadowGuard Pro compare l'adresse de retour de la pile standard avec celle de la Shadow Stack. Si elles ne correspondent pas, une alerte est générée et l'exécution est interrompue.
  • Détection des Attentats ROP : En empêchant la modification des adresses de retour, ShadowGuard Pro bloque les attaques ROP qui tentent de rediriger l'exécution du programme vers du code malveillant.
  • Intégration via ptrace : ShadowGuard Pro utilise l'interface ptrace pour lancer et surveiller l'application à sécuriser, permettant une protection transparente sans modification du code source.

5. Déploiement et Intégration

5.1. Prérequis

  • Système d'exploitation : Linux (versions supportées : Ubuntu, CentOS, Debian, etc.)
  • Architecture : x86_64
  • Droits administrateur : Nécessaires pour l'utilisation de ptrace

5.2. Installation

  1. Téléchargement : Téléchargez le paquet ShadowGuard Pro depuis notre site web.
  2. Installation : Suivez les instructions d'installation fournies dans le guide d'installation.
  3. Configuration : Configurez les paramètres de protection selon vos besoins.

5.3. Utilisation

ShadowGuard Pro est un exécutable qui lance et surveille l'application à sécuriser via ptrace. La syntaxe de base est la suivante :

shadowguard-pro [options]  [arguments]

Exemple :

shadowguard-pro --log-level=debug /usr/bin/mon_application --option1 --option2

5.4. Options Courantes

  • --log-level : Niveau de journalisation (debug, info, warning, error)
  • --output : Fichier de sortie pour les logs
  • --no-protect : Désactive la protection pour le débogage
  • --help : Affiche l'aide
